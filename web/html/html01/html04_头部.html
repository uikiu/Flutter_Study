html 头部

header元素包含了所有的头部标签元素。在header元素中可以插入类型
  • 脚本：script
  • 样式：css
  • 元：meta

可插入到header中的子标签

  • title
  • style
  • meata
  • link
  • script
  • noscript
  • base
base标签

base标签描述了基本的链接地址和链接目标，会作为html文档中所有链接标签的默认标签

link标签

link标签定义html文件与外部资源之间的关系，通常用于链接到外部的样式表css

style标签

style定义html文档的样式文件引用地址，在style中也可以直接添加样式来渲染html

meta标签

meta标签描述基本的元数据用于被浏览器解析

  • meta标签keywords:搜索引擎关键字
  • description：为网页定义描述内容
  • auth:网页作者
  • http-equiv="refresh"：网页刷新频率，秒
